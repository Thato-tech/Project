# Homelab

## Objective


The Homelab Lab project aimed to safely practice, stimulate and strengthen cybersecurity skills by building, attacking and defending systems in a controlled area. The primary focus was to provide a safe, hands on environment for learning, experimenting and practising cybersecurity and IT skills without risking real world-systems .

### Skills Learned

- Advanced understanding of networking .
- Proficiency in analyzing and interpreting network logs.
- Enhanced knowledge of network protocols.
- Development of critical thinking and problem-solving skills in cybersecurity.

### Tools Used

- Splunk used for Security Information and Event Management (SIEM) .
- Virual Box tool used a hypervisor.
- Kali Linux used as an attacker machine
